1.2 通过第三方工具软件（成大的校园网连接不了…）

  • putty

    这个就比较小巧，登录页面比较简单

  • MobaXterm

    这个页面比较好看，可以有多种方式登录。

  • 以上是我用过的，其他的详情可以参照此文章：Windows 下的免费 SSH 客户端工具

2. 文件传输的软件（可以帮助我们让windows的文件直接复制到服务器的文件夹里面）

  • WinSCP

    界面简单明了，支持拖拽文件进行传输，容易上手，也是免费的

  • FileZilla（使用人数最多）

    同时发送、接收多个文件，非常灵活且易于使用，也是免费的
