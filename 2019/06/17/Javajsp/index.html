jsp基础语法和内置对象

发布 : 2019-06-17 分类 : jsp 浏览 :

JSP

1. 指令

* 作用：用于配置JSP页面，导入资源文件
* 格式：
    <%@ 指令名称 属性名1=属性值1 属性1名=属性值2 ... %>
* 分类：
    1. page        配置JSP页面的
        * 格式：<%@ page contentType="text/html;charset=UTF-8" errorPage="error.jsp" language="java" %>
        * contentType : 等同于response.setContentType()
            1. 设置响应体的mime类型以及字符集
            2. 设置当前jsp页面的编码（只能是高级的IDE才能生效，低级工具需要设置pageEncoding属性设置当前页面的字符集编码）
        * import ： 导包（以后在jsp页面写java代码，也要导包，就相当于把jsp页面当成.java文件）
        * errorPage ： 当前页面发生异常后，会自动跳转到制定的错误页面
        * isErrorPage : 标识当前页面死否是错误页面.
            1.    true: 是，可以使用内置对象exception
            2.    false ： 否，不可以使用内置对象exception

    2. include  页面包含的。 导入页面的资源文件 
        * <%@include file="part.jsp"%>
    3. tagline    导入资源，可以用其他标签，jstl那些
        * <%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
        * prefix ： 前缀，自定义取别名

2. 注释

1.    html注释：
    <!-- -->:只能注释html代码片段
2.    jsp注释：
    <%-- --%>:可以注释所有

3. 内置对象

* 在jsp页面中不需要创建，直接使用的对象
* 一共有9个：
    变量名                     真实类型                    作用
    1.    pageContext             PageContext                  当前页面共享数据，还可以获取其他八个其他内置对象
    2.    request                 HttpServletRequest            一次请求访问的多个资源（转发）
    3.    session                 HttpSession                一次会话的多个请求间共享数据
    4.    application        ServletContext                    所有用户间共享数据
    5.    response             HttpServletResponse        响应对象
    6.    page                 Object                        当前页面(Servelet)的对象      this
       （也就是Servelet对象，指向this） //final java.lang.Object page = this;
    7.    out                     JspWriter                    输出对象，数据输出到页面上
    8.    config                 ServletConfig                Servelet的配置对象
    9.    exception             Throwable                    异常对象
