#Linux服务器安装JDK

1.下载jdk（开始很重要，不要忽略）

1.1 首先到官网下载的是jdk的二进制文件，也就是tar.gz的压缩包

1.2 jdk最好下载单数的
​ 奇数版本为BUG修正并全部通过检验的版本，官方强烈推荐使用这个版本。偶数版本包含了奇数版本所有的内容，以及未被验证的BUG修复，Oracle官方表示：除非你深受BUG困扰，否则不推荐您使用这个版本。

1.3 jdk9及后面的版本，都没有自带jre,即使你的jdk为11，但是java -version命令看的是jre版本

2. 使用winScp传输到Linux服务器

使用winScp连接到服务器后，直接将文件放到Liunx服务器上就ok，一般放在服务器根目录下的usr/local文件夹下面
1.2 使用putty连接服务器，进入文件夹，使用命令：tar -xzvf 文件名.tar.gz

3. 配置jdk环境

1.找根目录下的etc\profile文件

2.使用命令vi或者vim或者gedit（如果下载了的话）打开profile，然后按Esc键，输入 :wq 保存退出

3.在profile文件最下面添加：
​
export JAVA_HOME=/usr/local/jdk-11.0.3 //导入服务器jdk地址，下面就可以粘贴复制了
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib:$CLASSPATH
export JAVA_PATH=${JAVA_HOME}/bin:${JRE_HOME}/bin
export PATH=$PATH:${JAVA_PATH}
​
4.刷新资源：source profile（如果在etc目录中），在根目录下就输入 source etc/profile

4. 检查

输入java -version（最好是输入这个来查看你的jre是多少版本的）
