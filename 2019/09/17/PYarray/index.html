python中的序列

发布 : 2019-09-17 分类 : python 浏览 :

序列

序列的定义

  • 序列是一块用于存放多个值连续内存空间，并且按一定顺序排列，可以通过索引取值

Python中的序列

  1. 列表（list）
  2. 元组（tuple）
  3. 字典（dictionary）
  4. 集合（set）
  5. 字符串

序列中的常用方法

1.索引

语法：序列[下标] 
下标范围：
        正下标 0 —— 序列长度减一
        负下标 负的序列长度 —— -1

2.切片

语法：序列[start：end：step]
参数属性：
        start：起始值（包括） 注：第一个参数可省略，但是：要保留
        end：结束值（不包括）
        step：切片的步长，就是一次跳几个 注：省略以后，默认值为1

3.相加

语法：序列 + 序列

注意：只能是同类型的序列相加

4.相乘

使用方法：序列 * 序列 = 重复n次的序列

小技巧：字符串也可以直接相乘，比如 “s”*10 = ‘ssssssssss’

5.检查某个元素是否是序列成员

语法：value in sequence
参数属性：
        value：要检查的元素
        sequence：序列
        关键字：in
返回值： Boolean

6.序列长度

语法：len(sequence)
返回值：int

7.序列中最大(小)元素

语法：max(sequence)
语法：min(sequence)

8.其他方法

转换为列表：list（sequence）
转换为字符串：str（sequence）
转换为索引序列：enumerate(sequence) 注：类似于枚举：(0, '乔丹')
序列元素和：sum(sequence)
排序序列：sorted(iterable, cmp=None, key=None, reverse=False)
    iterable -- 可迭代对象。
    cmp -- 比较的函数，这个具有两个参数，参数的值都是从可迭代对象中取出，此函数必须遵守的规则为，大于则返回1，小于则返回-1，等于则返回0。
    key -- 主要是用来进行比较的元素，只有一个参数，具体的函数的参数就是取自于可迭代对象中，指定可迭代对象中的一个元素来进行排序。
    reverse -- 排序规则，reverse = True 降序 ， reverse = False 升序（默认）。

反转序列：reversed(sequence)

注意：序列中只能是同类型才能用

序列的种类详解

列表（list）

1.创建列表：
    1.使用赋值运算符直接创建列表：listname = [元素1 元素2,...,元素n]
    2.创建空列表：emptylist = []
    3.创建数值列表：listname = list(range(start,end,step))
      start:起始值（包括）
      end：结束值（不包括）
      step：步长
2.删除列表：
    del（sequence）
3.遍历列表
    1.直接使用for循环（只能输出元素值）
        for item in listname:                
    2.使用for循环和enumerate()函数
        for index ,item in enumerate(sequence):
4.添加新元素：
    列表.append（addValue）
5.插入新元素：
    列表.insert（index,addValue）
6.添加列表：
    列表.extend（sequence）
7.修改元素：
    列表[index] = value    
8.删除元素：
    列表.remove(元素值)
9.统计指定元素次数
    列表.count(Value)
10.获取指定元素下标
    列表.index(Value)
11.获取数值列表元素和
    sum(列表[,reAddValue])
        reAddValue : 数值列表总和再加上该值

列表推导式(快速生成想要的列表)

语法：list = [Expression for var in range]  # range也可以是list
例子1：[item * item for item in range(0,10,2)] = [0, 4, 16, 36, 64]
例子2：[x*y for x in range(1,5) if x > 2 for y in range(1,4) if y < 3]
        他的执行顺序是:
        for x in range(1,5)
            if x > 2
                for y in range(1,4)
                    if y < 3
                        x*y   # 这句最后执行的哦

元组（tuple）

  • 特点：无法增加元素，无法修改元素（直接修改不行）

    1.元组的创建
    1.使用赋值运算符直接创建元组(只有一个元素时，必须加个“，”，不然是字符串，不是元组)
        tuplename = (元素1，元素2，元素3...)
    2.创建空元组
        tuplename = ()
    3.创建数值元组
        tuplename = tuple(range(start,end,step))            

2.访问元组元素
    1.直接使用print()函数输出
    2.索引
        tuple[index]
    3.切片
        tuple[start:end]
            包括start，不包括end
3.修改元组元素
    可以通过 + 号得到新元组
4.元组推导式
    类似于列表，只是把[] 变为 ()

5.元组与列表的区别
      元组                  列表
    不可变序列                   可变序列
    支持切片操作                 支持切片操作
    元组访问速度快           列表访问速度慢
    元组可以作为字典键        不能作为字典的键

    字典（dictionary）

    1.字典的创建

    使用dict()方法
    例子：
    >>> dict(a='a', b='b', t='t') #注，没有引号     
    {'a': 'a', 'b': 'b', 't': 't'}
    >>> dict(zip(['one', 'two', 'three'], [1, 2, 3]))   # 映射函数方式来构造字典
    {'three': 3, 'two': 2, 'one': 1} 
    >>> dict([('one', 1), ('two', 2), ('three', 3)])    # 可迭代对象方式来构造字典
    {'three': 3, 'two': 2, 'one': 1}
直接创建：dictionary = {"key":"value",....}

    2.通过键值对访问

    dictionary[key]

    3.遍历字典

    dict.item() # 获取可遍历对象dict_items([('key1', value), ('key', value),...])
dict.keys() #获取所有键 dict_keys(['key1', 'key2',...])
dict.values() #获取所有值 dict_values([value1, value2,..])

    4.添加修改字典元素

    dictionary[key] = value

    5.字典推导式

    类似于列表推导式
例子：
{i*i : random.randint(10,100) for i in range(1,10)}
{1: 47, 4: 18, 9: 46, 16: 39, 25: 31, 36: 70, 49: 74, 64: 22, 81: 37}

    集合（set）

    集合的创建
    setname = {"value1","value2",222}
    setname = set() # 空集合，也可以把列表转换为集合
集合的修改
    setname.add(elmanet) #添加元素
    setname.remove("value") # 删除指定元素
    setname.pop()    #删除随机元素
    setname.clear()    #全删
>>>x = set('runoob')

集合的方法：
    >>> y = set('google')
    >>> x, y
    (set(['b', 'r', 'u', 'o', 'n']), set(['e', 'o', 'g', 'l']))   # 重复的被删除
    >>> x & y         # 交集
    set(['o'])
    >>> x | y         # 并集
    set(['b', 'e', 'g', 'l', 'o', 'n', 'r', 'u'])
    >>> x - y         # 差集
    set(['r', 'b', 'u', 'n'])
    >>> x ^ y         # 对称差集

    字符串

    字符串格式化

    使用%操作符
    '%[-][+][0][m][.n]'%exp

    例子：
        >>> template = '编号: %09d\t公司名称: %s \t 官网:http://www.%s.com' # 定义模板
        >>> item = ("7","百度","baidu")
        >>> print(template%item)
        >>>编号: 000000007    公司名称: 百度      官网:http://www.baidu.com
