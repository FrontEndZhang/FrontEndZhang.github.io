python中的函数和类

发布 : 2019-09-19 分类 : python 浏览 :

函数

函数的创建

def functionname(可选参数):
    '''提示信息（就是在写调用函数的时候，会提示你）'''
    [functionbody]
    [return]

关键字参数

参数的位置可以换    

定义时：def funtionname(参数名1，参数名2...):

使用时：(参数名1=value,参数名2=value) #参数位置可以换

参数默认值

def functionname(...,[parameter1 = default]):

例子：def fun(name="zjy",age=18,sex="nan"):

可变参数

一个 *：
    def funtionname(*parameter):

    例子（打印可变参数）：
    def fun(*change):
        for item in change:
            print(item)

俩个 * （这种形式可以接收任意多个以关键字参数赋值的实际参数，并将其放到一个字典中。）：
    def funtionname(**parameter):

    例子：
        # 定义了支持参数收集的函数
        def test(x, y, z=3, *books, **scores) :
            print(x, y, z)
            print(books)
            print(scores)
        test(1, 2, 3, "C语言中文网" , "Python教程", 语文=89, 数学=94)

        # 打印
        1 2 3
        ('C语言中文网', 'Python教程')
        {'语文': 89, '数学': 94}

匿名函数

result = lambda 参数1，参数2，...:expression

expression:不能出现for或while等
# 匿名函数不需要return来返回值，表达式本身结果就是返回值。

类

定义类

class Classname:
    '''类的帮助信息'''
    statement
ClassName : 采用驼峰式命名

创建类的实例

objectName = ClassName(有参传参)

创建init()方法和初始化

创建：     
    class Person:
        '''提示信息'''
        def __init__(self): # init方法中，self是必传参数，但是创建对象时，self不需要传，Python解释器自己会把实例变量传进去
            statement

    例子：
        class Person:
        '''我是人类'''
        def __init__(self,name,age):
                print(name)
                print(age)

        zjy = Person("张家义","18") #self参数不用传哦

        张家义
        18

初始化成员变量：

    def __init__(self,var1,var2):
        var

创建类的成员并访问

类中的方法：
    def functionName(self,parameterlist):
        block

    self:是必须的，跟类中的__init__一样，调用时不用出现这个参数

类中的属性：

    注意：类中的属性值是创建的对象中共享的！

    例子：
        class Person:
            '''我是人类'''
            name = "aaa"
            age = 99

            def __init__(self):

                    Person.age+=1
                    print(Person.age)

            def  printName(self):
                    print(Person.age)

        zjy = Person() #self参数不用
        zmy = Person()
        ygg = Person()

        输出：    
        100
        101
        102

访问限制

_varName:protected
    类本身和子类可以访问
__foo：private
    类本身可以访问
    但是可以通过:类的实例名._类名__xxx 获取该私有属性，且可以进行修改
    但是实例方法中不会采用这个修改的值
__foo__:系统

属性安全保护机制

@property：不能修改但是能读取
    注意：调用时，不能加括号，加了就错了
@属性名.setter:可以修改私有值

例子：

    class Goods:
         def __init__(self):
             self.age = 18

          @property
          def price(self): # 读取
            return self.age

          # 方法名.setter
          @price.setter # 设置,仅可接收除self外的一个参数
          def price(self, value):
            self.age = value

    obj = Goods()  # 实例化对象 
    obj.age  # 直接获取 age属性值
    obj.age= 123   #  修改age的值

类的继承

语法： class ClassName(FatherClass):

例子：
    class Person:
    '''我是人类'''
    def __init__(self):
            print("我是你爸")
    def father(self):
            print("我是father方法！")

    class Ygg(Person):
            '''我是狗袁帅'''
            def __init__(self):
                    super().__init__()
                    print("我是你儿")
            def son(self):
                    print("我是son方法！")
